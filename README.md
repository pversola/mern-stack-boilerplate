# MERN STACK Boilerplate

### What is MERN Stack

- MongoDB
- Express
- REACT
- NodeJs

### How to

git clone repository

```
$ git clone https://<URL>/<REPO_NAME>/<PROJECT_NAME>.git
$ cd <PROJECT_NAME>
```

### Script

Run Server

```
$ cd server
$ npm install
$ npm start
```

Run Client

```
$ cd client
$ yarn add
$ yarn dev
```
